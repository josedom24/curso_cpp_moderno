# Ejercicios con `std::array`

## Ejercicio 1: Mostrar los elementos del array

Declara un array de 5 enteros, pide al usuario que introduzca sus valores y, utilizando una función, muestra por pantalla el contenido del array, separando los elementos por espacios.

## Ejercicio 2: Calcular la suma y la media

Pide al usuario que introduzca los valores de un array de 6 números enteros. Crea una función que calcule y devuelva la suma de los elementos. Después, desde el programa principal, calcula y muestra la media.

## Ejercicio 3: Buscar un valor

Crea un array de 7 enteros, cuyos valores se introducen por teclado. Diseña una función que reciba el array y un valor a buscar, e indique si ese valor se encuentra dentro del array o no.

## Ejercicio 4: Encontrar el mayor y el menor valor

Declara un array de 8 números enteros. Los valores se introducen por teclado. Crea una función que determine el valor más pequeño y el valor más grande del array, y los muestre por pantalla.

## Ejercicio 5: Invertir el contenido del array

Crea un array de 10 enteros. Los valores se introducen por teclado. Diseña una función que modifique el array invirtiendo el orden de sus elementos y, desde el programa principal, muestra el resultado final.
