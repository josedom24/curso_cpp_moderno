# Ejercicios con `std::list`

## Ejercicio 1: Crear y mostrar una lista

Pide al usuario que introduzca varios números enteros y los almacene en una lista. El programa debe finalizar la introducción de datos cuando el usuario escriba un número negativo. Utilizando una función, muestra el contenido de la lista, separando los elementos por espacios.

## Ejercicio 2: Insertar elementos al principio y al final

Crea una lista de cadenas de texto vacía. Pide al usuario varios nombres. Si el nombre comienza por la letra "A" (mayúscula), debe insertarse al principio de la lista. En cualquier otro caso, se inserta al final. Muestra el contenido final de la lista.

## Ejercicio 3: Buscar un elemento en la lista

Pide al usuario que introduzca varios números enteros y los almacene en una lista. Luego, solicita un número y, utilizando una función, indica si dicho número se encuentra en la lista o no.

## Ejercicio 4: Eliminar elementos concretos

Diseña un programa que almacene varios números enteros en una lista. Después, solicita al usuario un valor. Crea una función que elimine todas las apariciones de ese valor en la lista. Muestra el resultado final.

## Ejercicio 5: Ordenar y mostrar la lista

Crea una lista de números enteros. Los valores se introducen por teclado. Una vez completada la lista, ordénala de menor a mayor y muestra el contenido ordenado.
