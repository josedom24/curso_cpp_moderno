# Ejercicios con `std::string`

## Ejercicio 1: Recorrido carácter a carácter

Escribe un programa que solicite una frase por teclado y, utilizando una función, muestre por pantalla cada uno de sus caracteres en líneas separadas, indicando además la posición que ocupa dentro de la cadena.

## Ejercicio 2: Contar apariciones de un carácter

Pide al usuario una cadena de texto y un único carácter (válida que sea un carácter). Crea una función que cuente cuántas veces aparece dicho carácter en la cadena y muestre el resultado.

## Ejercicio 3: Contar palabras en una frase

Solicita al usuario una frase por teclado que contenga varias palabras separadas por espacios. Mediante una función, determina cuántas palabras tiene la frase y muestra el resultado.

## Ejercicio 4: Invertir una cadena

Escribe un programa que lea una cadena de texto y, mediante una función, genere y muestre una nueva cadena que sea la inversión de la original.

## Ejercicio 5: Comprobar si es palíndromo

Diseña un programa que lea una palabra o frase por teclado e indique, usando una función, si se trata de un palíndromo (se lee igual de izquierda a derecha que de derecha a izquierda). El programa debe ignorar los espacios y no distinguir entre mayúsculas y minúsculas para esta comprobación.
