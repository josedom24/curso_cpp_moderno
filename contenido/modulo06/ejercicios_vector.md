# Ejercicios con `std::vector`

## Ejercicio 1: Crear y mostrar un vector

Crea un vector de números enteros. El programa debe pedir al usuario cuántos elementos desea introducir y luego leer los valores por teclado. Utilizando una función, muestra todos los elementos del vector, separados por espacios.

## Ejercicio 2: Calcular suma y media de un vector

Solicita al usuario que introduzca varios números enteros (hasta que escriba un valor negativo que no debe almacenarse). Almacena los valores en un vector. Crea una función que calcule y devuelva la suma de los elementos. Calcula y muestra la media desde el programa principal.

## Ejercicio 3: Buscar un valor en el vector

Crea un vector de cadenas de texto. Pide al usuario que introduzca varios nombres (hasta que escriba "fin"). Crea una función que reciba el vector y un nombre a buscar, e indique si ese nombre está en la lista.

## Ejercicio 4: Eliminar un elemento del vector

Partiendo de un vector de enteros que se llena con valores introducidos por el usuario, diseña una función que elimine la primera aparición de un valor concreto indicado por el usuario. Muestra el vector resultante desde el programa principal.

## Ejercicio 5: Ordenar un vector y mostrarlo

Crea un vector de números enteros. Los valores se introducen por teclado. Una vez llenado el vector, ordena sus elementos de menor a mayor y muestra el vector resultante. Utiliza una función para mostrar el vector.
